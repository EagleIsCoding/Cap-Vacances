# Cap-Vacances 🏖️

**Cap-Vacances** est une application web de réservation de logements de vacances développée avec le framework **Symfony**. Ce projet permet de consulter un catalogue de destinations, de gérer son compte utilisateur et d'effectuer des réservations en ligne de manière intuitive.

---

## 🚀 Fonctionnalités

* **🏠 Catalogue Dynamique** : Consultation des logements de vacances disponibles avec détails.
* **👤 Gestion Utilisateur** : Système complet d'inscription, de connexion et gestion du profil.
* **📅 Réservations** : Processus de réservation de séjours en ligne intégré.

---

## 🛠️ Stack Technique

* **Backend** : ![PHP](https://img.shields.io/badge/PHP-8.4.18-777BB4?style=flat-square&logo=php&logoColor=white) ![Symfony](https://img.shields.io/badge/Symfony-8.0.8-000000?style=flat-square&logo=symfony&logoColor=white)
* **Frontend** : Twig
* **Base de données** : MySQL / MariaDB
* **Environnement** : Machine Linux Distante

---

## 📁 Structure du projet

* **src/** : Contient toute la logique métier (Contrôleurs, Entités, Services).
* **templates/** : Fichiers Twig pour le rendu HTML.
* **public/** : Point d'entrée de l'application et assets publics.
* **config/** : Configuration du framework et des bundles.
